Community nodes

Ready-made blocks for ircuitry. Hit Copy, then paste the node straight into the app. Want to add your own? Open a PR.

Two ways to install:
  1. Install in app (one click): opens ircuitry and stages the node behind a confirm dialog. Needs ircuitry installed (Windows, macOS or Linux). On macOS, move ircuitry.app to Applications and open it once first.
  2. Copy, then in ircuitry open the Node Library and click Install from clipboard (or press Ctrl+Shift+V) and confirm. Works on every platform.
Prefer files? Use Download and drag the .ircnode onto the canvas, or drop it into ~/ircuitry/nodes/. To remove a node later, hover it in the Node Library and click the . Scripted nodes run code on your machine, so review one before installing.
