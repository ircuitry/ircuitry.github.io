Plugins

A plugin extends ircuitry itself - it adds menus, toolbar buttons, right-click actions, Cmd+K commands, its own docked side panel, dialogs and a settings screen - and works across every tab. Hit Install and ircuitry shows you exactly what it can do first. Built one? Open a PR.

How install works:
  1. One-click install: opens ircuitry and shows a trust card listing the plugin's permissions (side panel, runs commands, edits workflows...). Approve it and it installs + enables, on every tab. Needs ircuitry installed (Windows, macOS or Linux). On macOS, move ircuitry.app to Applications and open it once first.
  2. Download the .ircplugin and double-click it (or drop it on ircuitry) - same trust card.
A plugin you didn't approve never runs; disabling one sticks across restarts. Manage them in ircuitry under More -> Plugins.... Plugins can run sandboxed commands, use the network, or edit your workflows - the chips on each card (and the trust card) spell out exactly which.
