Drag blocks. Build anything.

ircuitry is a cozy, local-first studio where you wire visual blocks into real things: websites you eject as a clean React project, AI agents, desktop tools, everyday automations - and genuine IRC bots. Everything n8n does, plus everything it can't - and it all runs on your machine. No code required (but there's a Code node for when you want it).

One canvas, so much more than bots

A visual editor, a from-scratch IRCv3 client, a web compiler, AI, sandboxed code and a runtime - drawn by hand, all in one cozy app.

Visual node graph

Drag triggers, filters, actions and data nodes onto one canvas and wire them up. Exec flow plus typed data pins - what you build, you can read.

Build real websites

Design a page in nodes, preview it live, then eject a clean React + Vite project. Frontend, backend and data on a single canvas.

AI agents & code

Ask AI with tools and memory, a sandboxed Programmer AI that reads and edits whole codebases, and MCP to plug into anything.

Automate everything

HTTP, webhooks, sockets, databases, containers, schedules and calendars. Connect the apps you use - and reach what cloud tools can't.

Real IRCv3 bots

TLS, SASL, message tags, reactions and threads. A genuine from-scratch client, not a wrapper. Run many bots at once.

Local-first & yours

Self-contained, offline-capable, no accounts. Plugins extend the app itself, and you can eject your work anytime. Zero lock-in.

Beyond a workflow tool

Cloud automation tools wire SaaS APIs together on someone else's servers. ircuitry does that too, then keeps going: it builds the bot, the website and the desktop tool, runs the AI and the sandboxed code, and lives on your own machine - no accounts, no subscriptions, no servers to babysit. One cozy app you actually own.

Add powers from the community

Ready-made nodes (weather, dice, hashes, web APIs), whole workflows (games, reminders, even a website builder), plugins (a terminal, an AI assistant) and themes. Copy one and paste it straight into ircuitry.
